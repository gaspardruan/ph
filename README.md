# Ruan Zhongqiu's Home Website

Static personal site built with Astro, React islands, Tailwind CSS, and MDX content collections.

## Development

```bash
pnpm install
pnpm dev
```

## Checks

```bash
pnpm lint
pnpm build
```

## Deployment

The production build outputs static files to `dist/`. The Docker image serves them with Nginx.

```bash
./build.sh v1.0.0
```
